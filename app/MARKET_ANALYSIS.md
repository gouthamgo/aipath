## Complete Curriculum Structure

### Phase 1: Python Foundations (Free)
- Python Essentials (7 lessons)
- Data Manipulation (5 lessons)
- APIs & HTTP (5 lessons)

### Phase 2: LLM Fundamentals
- OpenAI API Mastery (8 lessons)
- LangChain Essentials (10 lessons)
- Prompt Engineering (9 lessons)
- Structured Outputs (7 lessons)
- Vector Databases (9 lessons)
- RAG Applications (10 lessons)
- **Multi-Provider LLMs** (7 lessons) - Claude, Gemini, LiteLLM
- **LLM Fine-Tuning** (8 lessons) - LoRA, QLoRA, Hugging Face

### Phase 3: AI Agents
- Agent Fundamentals (6 lessons)
- LangGraph (5 lessons)
- Multi-Agent Systems (5 lessons)
- **MCP & Tool Integration** (8 lessons) - Model Context Protocol

### Phase 4: Production AI
- LLMOps (5 lessons)
- Deployment (5 lessons)
- Advanced Patterns (5 lessons)
- **AI Testing & Evaluation** (7 lessons) - Golden datasets, LLM-as-judge
- **AI Safety & Guardrails** (6 lessons) - Prompt injection, moderation

### Phase 5: Specialized AI
- **Open Source LLMs** (7 lessons) - Ollama, Llama, Mistral
- **Multimodal AI** (7 lessons) - Vision, Whisper, multimodal RAG
- **Voice AI Agents** (7 lessons) - Speech recognition, synthesis
